## Contexte
Vous devez développer un système de supervision réseau qui surveille l'état des équipements
(serveurs, routeurs, switches) et génère des alertes en cas de problème.
## Objectifs
### Compétences techniques :
• Python (gestion fichiers, threading)
• Base de données MySQL (création, requêtes complexes)
• Réseau (ping, scan de ports, analyse)
• Automatisation et scheduling
• Génération de rapports
### Compétences métier ASSR :
• Surveillance d'infrastructure réseau
• Détection d'anomalies
• Gestion des incidents
• Documentation technique
